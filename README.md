# Beatsy
Drum computer using the Audio library

Inputs:
 - 8 Piezo triggers connected to Teensy
 - Adafruit trellis for selecting banks/tracks.

What works?
 - 4 Piezo triggers
 - Playing samples polyphonically with velocity
 - Multiplexer for analog inputs
